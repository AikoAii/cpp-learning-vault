# Learning Strategy

## Do
- fokus konsep
- pahami ownership
- gunakan STL
- review rutin
- buat internal linking

## Avoid
- menghafal syntax
- terlalu cepat pindah topik
- mengabaikan memory model
- menggunakan raw pointer sembarangan
